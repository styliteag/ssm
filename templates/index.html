Welcome to Secure SSH Manager!

Your centralized solution for managing SSH public keys across multiple servers. Keep your infrastructure secure with automated key deployment and monitoring.

Quick Start

🖥️ Manage Hosts

Add new servers, view existing hosts, and configure SSH connections.

👥 Manage Users

Create users, assign SSH keys, and control access permissions.

🔑 Manage Keys

Upload, organize, and deploy SSH public keys across your infrastructure.

🔍 View Issues

Monitor key differences and resolve configuration conflicts.

Key Features

  • Centralized Management: Control SSH keys for all your servers from one place
  • Automated Deployment: Push key changes to multiple hosts simultaneously
  • Diff Monitoring: Track changes and identify configuration drift
  • User Permissions: Fine-grained access control per host and user
  • Audit Trail: Keep track of who has access to what and when

Getting Started

  1. Add a Host: Start by adding your first server to manage
  2. Create Users: Set up user accounts for your team members
  3. Upload Keys: Add SSH public keys for authentication
  4. Assign Access: Link users and keys to specific hosts
  5. Deploy & Monitor: Push changes and monitor for issues
